# The Tiled Archive

A curated personal knowledge garden built with performance and simplicity in mind. This repository hosts the source code for [umitso.com](https://umitso.com).

## Tech Stack

*   **Framework:** [Next.js](https://nextjs.org/) (App Router)
*   **Styling:** [Tailwind CSS](https://tailwindcss.com/)
*   **Deployment:** GitHub Pages (Static Export)

## Setup Instructions

1.  **Install Dependencies:**
    ```bash
    npm install
    ```

2.  **Run Development Server:**
    ```bash
    npm run dev
    ```

3.  **Build for Production:**
    ```bash
    npm run build
    ```

## Deployment Status

This project is automatically deployed to [umitso.com](https://umitso.com) via GitHub Actions.
